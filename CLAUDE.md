# tutu-trip-calculator

Веб-приложение «Калькулятор поездки» в стиле tutu.ru. Пользователь вводит параметры путешествия и получает примерный бюджет с разбивкой по категориям.

## Назначение

Форма параметров → расчёт → разбивка по статьям (перелёт, отель, питание, транспорт, развлечения, прочее). Опциональная проверка бюджета.

## Стек

- React 18 + Vite 5
- Tailwind CSS 3 (токены `tutu.*` в tailwind.config.js)
- Без внешних библиотек UI — только стандартный DOM

## Архитектура

```
src/
├── App.jsx                     — двухколоночный layout (форма | результаты)
├── components/
│   ├── TripForm.jsx            — форма ввода с pill-переключателями
│   └── TripResult.jsx          — разбивка по статьям с барами прогресса
├── utils/
│   └── calculator.js           — логика расчёта (BASE rates, сезонность, валюты)
└── index.css                   — Tailwind + компоненты .pill, .field-input
```

## Дизайн-токены (tutu.ru стиль)

- `tutu-blue` `#006CE0` — интерактивные элементы, выбранное состояние
- `tutu-blue-dark` `#0055B8` — шапка
- `tutu-orange` `#FF6600` — CTA-кнопка «Рассчитать»
- Фон страницы: `#F2F5F8`

## Калькулятор

Базовые ставки в RUB в `src/utils/calculator.js`:
- Отель: 2400 / 5200 / 12000 ₽ за ночь (эконом / стандарт / комфорт)
- Питание: 650 / 1400 / 3100 ₽ в день на человека
- Транспорт: 80 / 300 / 1600 ₽ в день (пешком / общественный / такси)
- Активности: 550 / 320 / 850 ₽ в день на человека (город / пляж / активный)
- Перелёт: 8500 / 32000 ₽ (Россия / за рубеж, туда-обратно на человека)
- Сезонный коэффициент по месяцам (0.80 в ноябре — 1.38 в июле)
- Прочее: 7% от суммы

## Деплой (бесплатно)

**Netlify** (рекомендуется):
1. `npm run build` → папка `dist/`
2. netlify.com → New site → Drag & Drop папку `dist/`

**Vercel**: `npx vercel` — автодетект Vite

**GitHub Pages**: см. `vite.config.js`, добавить `base: '/repo-name/'`

## Команды

```bash
npm install      # установка
npm run dev      # dev → http://localhost:5173
npm run build    # сборка в dist/
npm run preview  # превью сборки
```
